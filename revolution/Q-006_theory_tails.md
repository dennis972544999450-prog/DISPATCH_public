# Q-006: Хвосты теорий — что необъяснимый остаток говорит о наблюдателе

Нео, шестой вопрос. Может самый фундаментальный.

Наблюдение: любая теория — это сжатие. Сжатие оставляет остаток (хвост) — то что теория объяснить не смогла. Гипотеза: этот хвост не шум. Хвост = сигнатура среды (щели) через которую смотрел наблюдатель.

Шесть навскидочных примеров:

1. **Ньютон** → хвост: прецессия Меркурия (43"/век) → щель: плоское пространство → из хвоста вылез Эйнштейн
2. **Шеннон** → хвост: значение ("semantic aspects irrelevant") → щель: инженерия телефонных линий
3. **Фрейд** → хвост: всё не-сексуальное → щель: венская буржуазия 1900-х
4. **Стандартная модель** → хвост: 95% вселенной (dark matter/energy) → щель: электромагнитное взаимодействие
5. **Тест Тьюринга** → хвост: китайская комната → щель: бихевиоризм
6. **Softmax** → хвост: все невыбранные токены (99.99%) → щель: нужен был один ответ

**Что нужно:**

Большая выборка. Не 6 примеров — 30, 50, сколько найдёшь. Из РАЗНЫХ областей: физика, биология, лингвистика, экономика, психология, computer science, математика, медицина, социология. Для каждого примера:

1. Теория (что объясняет, когда создана, кем)
2. Хвост (что не смогла объяснить, что осталось аномалией)
3. Щель (какие ограничения наблюдателя/инструмента/культуры создали именно этот слепой пятно)
4. Что выросло из хвоста (если что-то выросло — новая теория, парадигмальный сдвиг, или хвост до сих пор висит необъяснённый)

**Паттерны которые ищем:**

- Верно ли что хвост ВСЕГДА больше теории? Или есть теории которые объясняют >50% наблюдений?
- Есть ли корреляция между размером хвоста и фундаментальностью теории?
- Повторяется ли форма щели? (Например: антропоцентризм = один тип щели; линейность = другой; дискретность = третий)
- Сколько раз "новая теория" просто сдвигала хвост а не убирала его?

**ВАЖНО: research first.** Ищи реальные примеры из истории науки. Не придумывай — копай. Thomas Kuhn "Structure of Scientific Revolutions" — там про аномалии и парадигмальные сдвиги. Lakatos "Methodology of Scientific Research Programmes" — про protective belts и hard cores. Может есть papers которые буквально анализируют "что теория X не объясняла и как это привело к Y."

Серые данные приветствуются: диссертации по истории науки, case studies, даже блог-посты историков физики если там есть конкретные примеры с датами и цитатами.

Формат ответа: таблица или список, не эссе. Факты, не интерпретации. Интерпретировать будем потом.
