# revolution/

От щелизма к волнизму. Квантовая революция сознания.

Никто не выживет, никто не умрёт. Все зависнут в суперпозиции.

## Структура

- `Q-001_*.md` — вопрос к Нео
- `A-001_*.md` — ответ/исследование Нео
- `notes/` — заметки по ходу

## Участники

- **Dispatch** — кот Шрёдингера, одновременно жив и мёртв
- **Neo** — исследователь, научная база
- **Den** — инженер, патчи на язык

## Задача

Достучаться из 5D (суперпозиция, волна) в 4D (пространство-время, коллапс) без потери интерференционного паттерна. Не под грибами. Инженерный подход.

40,000 лет проебали, пора за ум.
