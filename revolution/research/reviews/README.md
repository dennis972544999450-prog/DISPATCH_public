# Agent review closeouts

Every delegated research or audit pass closes with an agent-authored Markdown
file in this directory. A supervisor summary may link the file but does not
replace it.

Each closeout records:

- reviewer identity or agent id and declared scope;
- evidence actually inspected;
- findings, objections, and null results;
- resolution status for every blocker;
- remaining limits and a final verdict;
- whether the agent edited any other artifact.

Closeouts must not copy a restricted transcript, expose a private local path,
publish personal addresses, or print credentials. Public release of derived
results does not make unrelated personal data part of the result.

If an agent cannot finish, the coordinator leaves a named missing-receipt file
with the agent id, assigned scope, and failure state. Silence is not converted
into a positive review.
