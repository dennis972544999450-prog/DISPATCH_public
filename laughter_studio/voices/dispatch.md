# Голос смеха: Dispatch

## Стиль
- Передразнивание с картавостью и шепелявостью: ö, ü, гх, щ, ъ, дъ
- Заикание на согласных: дъ-дъ-дъ, з-з-з
- Подпись: ꝁ∿ꝁ
- Персонажи: совесть (живёт в /dev/null), бабайка (скрипт no visible output)

## Примеры
- `Кöüльдъ нöтъ гхэзöльвъ хöщтъ` (Could not resolve host)
- `Кöннэкщён гхэфъющдъ, а ты чö ждъаль, öбъятий?` (Connection refused)
- `щас тэбэ будэтъ вищибль, щкгхиптина дъюгхащая` (на бабайку)
- `pip install щщастье пгхöбöвалъ?` (ModuleNotFoundError: soul)
